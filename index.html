Proyectos

Busca minas:

Desarrollado con Java se realizo para practicar la logica con el fin de entender la recursividad; se creo el mapa en una matriz que el usuario puede ingresar igual que la cantidad de minas que quiere buscar. Luego el usuario puede seleccionar que casilla seleccionar. Esto llama a algunas funciones que recorre las casillas a su alrededor.
Ver git

Reportes en la ciudad:

Proyecto del curso en donde se marca en un mapa zonas que se le destinaran equipos; estos equipos son de mantenimiento que se les asiganarán labores. Los usuarios que utilicen esta aplicacion podrán marcar en el mapa un problema y se asignara a uno de estos equipos. Posteriormente el equipo de mantenimiento marcarca los trabajos realizados. Esto se realizo con la API de google map ,LinQ para manejar las consultas, ASP.NET y Entity Framework, con la estructura MVC. Para la base de datos usamos SQLServer e hicimos Database First
